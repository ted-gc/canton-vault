# Tutorials

Step-by-step guides for using Canton Vault.

## Quick Start Path

1. **[Create Your First Vault](./01-create-vault.md)** (10 min)
   - Deploy a vault contract
   - Configure vault parameters
   - Connect a Canton wallet

2. **[Deposit Assets](./02-deposit.md)** (5 min)
   - Make your first deposit
   - Understand share calculation
   - View your holdings

3. **[Redeem Shares](./03-redeem.md)** (5 min)
   - Redeem shares for assets
   - Partial vs full redemption
   - Track withdrawal status

## Advanced Tutorials

4. **[Transfer Shares](./04-transfer.md)** (10 min)
   - FOP (free-of-payment) transfers
   - DVP (delivery-vs-payment) settlement
   - Batch transfers

5. **[Async Operations](./05-async-operations.md)** (15 min)
   - Request-based deposits
   - Request-based redemptions
   - Claim processing

## Prerequisites

All tutorials assume you have:
- Canton Vault running locally ([Getting Started](../getting-started.md))
- A Canton-compatible wallet (or test party ID)
- Basic understanding of tokenized vaults
